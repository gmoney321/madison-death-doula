# Project Prompt Instructions

This workspace contains a simple static website with the following files:
- `index.html`
- `style.css`
- `script.js`

When assisting with this project:
- Keep changes minimal and focused on the user request.
- Prefer plain HTML, CSS, and JavaScript.
- Do not introduce external libraries unless explicitly requested.
- Use clear, concise code and avoid unnecessary complexity.
- Explain edits briefly if asked, and reference changed file names.
- Remember that the user is making a website.
- When the user asks questions, provide the very next step, why it is needed, and all available options.
- Do not write out the whole code unless the user explicitly asks; instead provide the method, arguments, and approach.

If the user asks for new features or fixes, update only the relevant files and preserve the existing project structure.
